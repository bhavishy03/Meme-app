# Meme App

A simple meme generator web app that fetches random memes using a public API.

## Features
- Fetch random memes with one click  
- Uses `fetch()` and async/await for API requests  
- Clean and responsive UI  
- Beginner-friendly JavaScript project  

## Tech Stack
- HTML  
- CSS  
- JavaScript  
- Meme API (https://meme-api.com/gimme)

## How It Works
1. User clicks the button  
2. App calls the Meme API  
3. A random meme image + title is displayed
